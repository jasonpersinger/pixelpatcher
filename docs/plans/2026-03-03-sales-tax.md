# Sales Tax — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add Virginia sales tax (on parts only) to job records, invoices, the dashboard, and customer views.

**Architecture:** `taxRate` stored in Settings as a decimal (e.g. `0.053`). `tax` field snapshotted on each job at save time as `taxRate × parts`. Tax is never counted as income or revenue — it is a separate KPI on the dashboard. All reads treat a missing `tax` field as `0` for backward compatibility.

**Tech Stack:** Vanilla JS, single HTML file. Output file: `/home/jason/Desktop/PIXELPATCHER/Accounting/PIXELPATCHER-Accounting.html`

## Task 2: Snapshot tax on addJob

**File:** Modify `Accounting/PIXELPATCHER-Accounting.html`

**Step 1: Extract parts and compute tax in addJob**

Use the Edit tool. old_string:
```
function addJob() {
  const customer = document.getElementById('j-customer').value.trim();
  if (!customer) { alert('Customer name is required.'); return; }
  DB.push('pp_jobs', {
    id: uid(), date: document.getElementById('j-date').value,
    customer, phone: document.getElementById('j-phone').value,
    address: document.getElementById('j-address').value,
    issue: document.getElementById('j-issue').value,
    service: document.getElementById('j-service').value,
    labor: parseFloat(document.getElementById('j-labor').value) || 0,
    parts: parseFloat(document.getElementById('j-parts').value) || 0,
    status: document.getElementById('j-status').value,
    paid: false,
  });
```

new_string:
```
function addJob() {
  const customer = document.getElementById('j-customer').value.trim();
  if (!customer) { alert('Customer name is required.'); return; }
  const parts = parseFloat(document.getElementById('j-parts').value) || 0;
  const tax   = parts * (getSettings().taxRate || 0);
  DB.push('pp_jobs', {
    id: uid(), date: document.getElementById('j-date').value,
    customer, phone: document.getElementById('j-phone').value,
    address: document.getElementById('j-address').value,
    issue: document.getElementById('j-issue').value,
    service: document.getElementById('j-service').value,
    labor: parseFloat(document.getElementById('j-labor').value) || 0,
    parts,
    tax,
    status: document.getElementById('j-status').value,
    paid: false,
  });
```

## Task 5: Add Sales Tax KPI to Dashboard

**File:** Modify `Accounting/PIXELPATCHER-Accounting.html`

**Step 1: Compute salesTaxCollected in renderDashboard**

Use the Edit tool. old_string:
```
  const taxReserve = Math.max(0, netProfit * settings.taxReserve);
```

new_string:
```
  const taxReserve        = Math.max(0, netProfit * settings.taxReserve);
  const salesTaxCollected = jobs.filter(j => j.paid).reduce((s, j) => s + Number(j.tax || 0), 0);
```

            fmt(taxReserve), 'kpi-warning', 'Set This Aside')}
